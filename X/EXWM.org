EXWM and polybar.

* EXWM

A full-featured [[https://github.com/ch11ng/exwm][tiling X window manager]] for Emacs. Example configuration [[https://github.com/ch11ng/exwm/wiki/Configuration-Example][here]].

#+begin_src emacs-lisp
(use-package exwm
  :config (exwm-enable))
#+end_src

** Global keys

Executing programs.

#+begin_src emacs-lisp
(defun sb/start-process (command)
  (start-process-shell-command command nil command))

(defun sb/execute-application (command)
  (interactive (list (read-shell-command "$ ")))
  (sb/start-process command))

(defun sb/exwm-add-global-key (cell)
  (let ((key (kbd (car cell)))
        (fun (cdr cell)))
    (when (stringp fun)
      (setq fun
            `(lambda ()
               (interactive)
               (sb/start-process ,fun))))
    `(,key . ,fun)))

(defun sb/exwm-add-global-keys (keys)
  (dolist (key keys)
    (push (sb/exwm-add-global-key key) exwm-input-global-keys)))

(sb/exwm-add-global-keys '(;; super prefix
                           ("s-r" . exwm-reset)
                           ("s-p" . sb/execute-application)
                           ("s-w" . "firefox")
                           ("s-t" . sb/ansi-term-bash)
                           ;; laptop keys
                           ("<XF86AudioMute>" . "amixer set Master toggle")
                           ("<XF86AudioLowerVolume>" . "amixer set Master 5%-")
                           ("<XF86AudioRaiseVolume>" . "amixer set Master 5%+")
                           ("<XF86AudioMicMute>" . "amixer set Capture nocap")
                           ("<XF86MonBrightnessDown>" . "brightnessctl --min-value=10 s 15%-")
                           ("<XF86MonBrightnessUp>" . "brightnessctl --min-value=10 s 15%+")))
#+end_src

      '(;; movement

* Polybar

A fast and easy-to-use [[https://github.com/polybar/polybar][tool for creating status bars]].

** Face

Designed to fit in the right part of Emacs' echo area.

#+begin_src conf
[bar/poly]
override-redirect = true
bottom = true
fixed-center = false

width = 20%
offset-x = 80%
height = 15

foreground = #ffffff
background = #282a36

font-0 = Fira Code:pixelsize=11:semibold;1

padding = 1
module-margin = 2
modules-right = pulseaudio bluetooth battery date
#+end_src

** Modules

Volume and bluetooth.

#+begin_src conf
[module/pulseaudio]
type = internal/pulseaudio

label-volume = "%percentage%%  "
label-muted = %percentage%% m

[module/bluetooth]
type = custom/script

exec = sh ~/.config/polybar/bluetooth
#+end_src

Print =b= if bluetooth headphones are connected.

#+begin_src shell :tangle ~/.config/polybar/bluetooth
if [ $(bluetoothctl info | wc -l) -ne 1 ]; then
    echo "b"
else
    echo " "
fi
#+end_src
